# validateBrackets

Given a string containing only `(`, `)`, `{`, `}`, `[` and `]`, determine if the given string is valid.

A string is valid if it is closed by the same type of brackets and closed in correct order.

### Examples
* input `()` returns `true`
* input `(){}[]` returns `true`
* input `(}` returns `false`
* input `({)}` returns `false`
* input `{[]}` return `true`
